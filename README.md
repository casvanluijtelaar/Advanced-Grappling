# Advanced Grappling

**Advanced Grappling** is a mod for Arma 3 that adds throwable grappling hooks and launchers. It is an extension of the *Advanced Urban Rappelling* mod.


## Features

*   Throwable grappling hooks.
*   Grappling hook launchers.
*   Support for ACE throwing

## Requirements

*   Arma 3
*   [CBA_A3](https://steamcommunity.com/workshop/filedetails/?id=450814997)
*   [Advanced Urban Rappelling](https://steamcommunity.com/workshop/filedetails/?id=730310357)

## Installation

1.  Subscribe to the required mods on the Steam Workshop.
2.  Subscribe to this mod on the Steam Workshop (link to be added).
3.  Enable all mods in the Arma 3 launcher.

## For Developers

This project uses HEMTT for building.

*   Run `hemtt build` to create a development build.
*   Run `hemtt build --release` to create a release build.
*   Run `hemtt launch` to test without ACE, or `hemtt launch ace` to test with ACE.
